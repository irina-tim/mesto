# Проект: Место

**Посмотреть проект на GitHub Pages**

[Ссылка на страницу на GitHub Pages](https://irina-tim.github.io/mesto/index.html)

**Используемые технологии**

HTML, CSS, СSS animations, flexbox, grid, nested BEM, vanilla JavaScript
